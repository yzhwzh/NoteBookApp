# 3. 假设检验

### **假设检验基本思想**

理论根据是 **实际统计推断原理（小概率原理）**：即认为概率很小的事件在一次实验中几乎不会发生。在检验中，一般把概率不超过0.10,0.05,0.025,0.005或0.001等的事件，称为小概率事件。

基本思想方法是：具有概率性质的反证法，为了检验$$H_0$$是否成立，先假定其成立，在此条件下，如果产生了小概率事件，则说明假设$$H_0$$不正确，需否定该假设，反之则认为$$H_0$$正确，接受该假设。

步骤： 

* 建立假设，提出一个关于总体分布的命题，即确定原假设$$H_0$$与备择假设$$H_1$$。 
* 寻找检验统计量：要检验$$H_0$$是否为真，关键是要找小概率事件A，使得$$P(A) \le \alpha$$是小概率事件,$$\alpha$$是显著水平。

> 例如 $$H_0: \mu=\mu_0, H_1: \mu \ne \mu_0$$, 其中$$\mu$$的无偏估计是$$\overline X$$,如果让$$H_0$$成立，即应有$$P\{|\overline X - \mu|< C\} = 1-\alpha$$ 是大概率事件，进而$$P\{|\overline X - \mu|> C\} = \alpha$$是小概率事件。因$$\overline X \sim N(\mu,\sigma^2/n)$$，假如$$\sigma^2$$已知，带入$$H_0$$条件，此时$$\frac{\overline X - \mu_0}{\sigma_0/\sqrt{n}} \sim N(0,1)$$ 既是寻找的检验统计量。

* 检验的拒绝域：拒绝原假设的样本观测值所组成的区域，反之即是接受域。 例$$P\{|\overline X - \mu|> C\}$$就是要寻找的小概率事件，其$$P\{|\overline X - \mu|> C\} = P\{\frac{|\overline X - \mu|}{\sigma_0/\sqrt{n}}>\frac{C}{\sigma_0/\sqrt{n}}\} = \alpha$$, **注意，此处为=号，若是单侧检验，是** $$P<=alpha$$,采用双侧分位数，则$$\frac{C}{\sigma_0/\sqrt{n}} = Z_{\alpha/2} \Rightarrow C= \frac{\sigma_0}{\sqrt{n}}Z_{\alpha/2}$$所以拒绝域为$$W = \{ (x_1,x_2,...,x_n):|\overline x-\mu_0| > \frac{\sigma_0}{\sqrt{n}}Z_{\alpha/2}\}$$或 $$\{ (x_1,x_2,...,x_n):\frac{|\overline X - \mu|}{\sigma_0/\sqrt{n}}>\frac{C}{\sigma_0/\sqrt{n}}\}$$, $$Z_{\alpha/2}$$ 为检验的临界点， 位于该区域的观测值，则拒绝原假设。
*  查表，计算，判断

**两类错误**

如果实际真实情况$$H_0$$为真，而由于样本值落入拒绝域而做出拒绝$$H_0$$的错误言论，称为拒真错误，反之，则是存伪错误。 两类错误此消彼长，但总是遵循控制犯第一类错误的概率，因为其往往造成的损失更大。

### **假设检验问题的类型**

1. 参数检验（总体均值，均值差的检验；总体方差，方差比的检验）
2. 非参数检验（分布拟合检验，符号检验，秩和检验）

### **参数的假设检验**

#### **1. 一个正态总体的均值假设检验**

a. 已知方差$$\sigma^2 = \sigma_0^2 $$ ，假设为$$H_0: \mu=\mu_0, H_1: \mu \ne \mu_0$$。  
检验统计量$$\frac{\overline X - \mu_0}{\sigma_0/\sqrt{n}} \sim N(0,1)$$

给定显著水平$$\alpha$$, $$H_0$$拒绝域为$$\frac{|\overline X - \mu_0|}{\sigma_0/\sqrt{n}}> Z_{\alpha/2}$$

由于这种检验法所选取的检验统计量是正态分布，故称为U-检验法

b. 方差未知  
检验统计量：$$\frac{\overline X - \mu_0}{S/\sqrt{n}} \sim t(n-1)$$

拒绝域：$$\frac{|\overline X - \mu_0|}{S/\sqrt{n}}> t_{\alpha/2}(n-1)$$

c. 方差已知$$\sigma^2 = \sigma_0^2 $$，假设为$$H_0: \mu \le \mu_0, H_1: \mu \gt \mu_0$$

此时$$P\{\overline X - \mu_0> C\} $$是小概率事件

并且$$\frac{\overline X - \mu}{\sigma_0/\sqrt{n}} \ge\frac{\overline X - \mu_0}{\sigma_0/\sqrt{n}}\Rightarrow \alpha = P\{\frac{\overline X - \mu}{\sigma_0/\sqrt{n}}>C\} \ge P\{\frac{\overline X - \mu_0}{\sigma_0/\sqrt{n}}>C\}$$

即$$\frac{\overline X - \mu_0}{\sigma_0/\sqrt{n}}>Z_\alpha$$

是一个概率不大于$$\alpha$$的事件，即是拒绝域，此为右侧检验。 反之$$H_0: \mu \ge \mu_0, H_1: \mu \lt \mu_0$$； $$P\{\overline X - \mu_0<- C\}$$  是小概率事件； $$\frac{\overline X - \mu}{\sigma_0/\sqrt{n}} \le\frac{\overline X - \mu_0}{\sigma_0/\sqrt{n}}\Rightarrow \alpha = P\{\frac{\overline X - \mu}{\sigma_0/\sqrt{n}}<-C\} \ge P\{\frac{\overline X - \mu_0}{\sigma_0/\sqrt{n}}<-C\}$$ 

$$\frac{\overline X - \mu_0}{\sigma_0/\sqrt{n}}<-Z_\alpha$$

为拒绝域

d. 方差未知， 假设为$$H_0: \mu \le \mu_0, H_1: \mu \gt \mu_0$$ 拒绝域为

$$\frac{\overline X - \mu_0}{S/\sqrt{n}}> t_{\alpha}(n-1)$$

反之拒绝域为：

$$\frac{\overline X - \mu_0}{S/\sqrt{n}}<-t_{\alpha}(n-1)$$

#### **2. 一个正态总体的方差假设检验**

a. 未知$$\mu$$， 假设为$$H_0: \sigma^2 = \sigma_0^2, H_1: \sigma^2 \ne \sigma_0^2$$

检验统计量为

$$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$

拒绝域为：$$0<\frac{(n-1)S^2}{\sigma_0^2}<\chi_{1-\alpha/2}^2(n-1) or \frac{(n-1)S^2}{\sigma_0^2}>\chi_{\alpha/2}^2(n-1)$$

此为$$\chi^2$$检验法

a.1. 假设为$$H_0: \sigma^2 <= \sigma_0^2, H_1: \sigma^2 > \sigma_0^2$$  
拒绝域为:$$\chi^2 > \chi_{\alpha}^2(n-1)$$  
a.2. 假设为$$H_0: \sigma^2 >= \sigma_0^2, H_1: \sigma^2 < \sigma_0^2$$  
拒绝域为:$$\chi^2 < \chi_{1-\alpha}^2(n-1)$$

b. 已知$$\mu$$， 检验统计量为

$$\frac{\sum_{i=1}^n(X_i-\mu)^2}{\sigma^2} \sim \chi^2(n)$$

拒绝域为：

$$0<\frac{\sum_{i=1}^n(X_i-\mu)^2}{\sigma_0^2}<\chi_{1-\alpha/2}^2(n) or \frac{\sum_{i=1}^n(X_i-\mu)^2}{\sigma_0^2}>\chi_{\alpha/2}^2(n)$$

b.1. 假设为$$H_0: \sigma^2 <= \sigma_0^2, H_1: \sigma^2 > \sigma_0^2$$  
拒绝域为:$$\chi^2 > \chi_{\alpha}^2(n)$$  
b.2. 假设为$$H_0: \sigma^2 >= \sigma_0^2, H_1: \sigma^2 < \sigma_0^2$$  
拒绝域为:$$\chi^2 < \chi_{1-\alpha}^2(n)$$

#### **3. 两个正态总体的均值差，方差比假设检验**

a. 两个正态总体方差的假设检验

未知$$\mu_1,\mu_2$$; 

检验统计量$$S_1^2/S_2^2 \sim F(m-1,n-1)$$

双侧拒绝域

$$S_1^2/S_2^2 > F_{\alpha/2}(m-1,n-1) or S_1^2/S_2^2 < F_{1-\alpha}(m-1,n-1)$$

单侧拒绝域

$$H_0:\sigma^2 <= \sigma_0^2;  S_1^2/S_2^2 > F_{\alpha}(m-1,n-1)$$

$$H_0:\sigma^2 >= \sigma_0^2;  S_1^2/S_2^2 < F_{1-\alpha}(m-1,n-1)$$

已知$$\mu_1,\mu_2$$; 此时，**标准差计算减去的是**$$\mu$$**，不是**$$\overline X$$; 服从分布的自由度变为$$F(m,n)$$, 其余不变。

b. 两个正态总体均值差的假设检验

* 已知方差 $$H_0:\mu_1 = \mu_2; \mu_1 \ne \mu_2$$ 

检验统计量：$$\frac{\overline{X}-\overline{Y}-(\mu_1-\mu_2)}{\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}} \sim N(0,1)$$

带入条件：$$\frac{\overline{X}-\overline{Y}}{\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}} \sim N(0,1)$$

拒绝域为：$$\frac{\overline{X}-\overline{Y}}{\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}} > Z_{\alpha/2}$$

单侧检验拒绝域为：

$$H_0 :\mu_1 >= \mu_2:\frac{\overline{X}-\overline{Y}}{\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}} > Z_{\alpha}$$

$$H_0:\mu_1 <=\mu_2:\frac{\overline{X}-\overline{Y}}{\sqrt{\frac{\sigma_1^2}{n_1}+\frac{\sigma_2^2}{n_2}}} < -Z_{\alpha}$$

* 未知方差，有方差齐性的条件：

$$\frac{\overline{X}-\overline{Y}-(\mu_1-\mu_2)}{S_W\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}} \sim t(n_1+n_2-2)$$

$$S_W = \sqrt{\frac{(n_1-1)S_1^2+(n_2-1)S_2^2}{n_1+n_2-2}}$$

拒绝域：$$|T| > t_{\alpha/2}(n_1+n_2-2)$$

同理，单侧检验：

$$H_0 :\mu_1 >= \mu_2: T > t_{\alpha}(n_1+n_2-2)$$

$$H_0 :\mu_1 <= \mu_2: T < -t_{\alpha}(n_1+n_2-2)$$

**在没有告知是否有方差齐性的条件，需要先判断是否方差齐性，只有在齐性的条件下才能用**$$t$$**检验法检验均值是否相等；**$$t$$**检验的前提条件是两样本独立、正态、方差齐。满足正态但方差不齐的用**$$t'$$**检验，不正态的用秩和检验**

c. 成对样本数据的假设检验：在相同条件下，作比对试验的数据

设$$X，Y$$相互独立，服从正态分布，则$$Z=X-Y \sim N(\mu_1-\mu_2,\sigma_1^2+\sigma_2^2)$$

检验假设$$H_0: \mu = \mu_1-\mu_2 = 0, H_1: \mu \ne 0$$

检验统计量为：$$\frac{\overline Z-\mu}{S/\sqrt{n}} \sim t(n-1)$$

当$$H_0$$成立时：$$\frac{\overline Z}{S/\sqrt{n}} \sim t(n-1)$$

故拒绝域$$\frac{|\overline Z|}{S/\sqrt{n}} > t_{\alpha/2}(n-1)$$

单侧检验同理

d. **总体比例的检验** 离散 $$X$$服从$$0-1$$分布

若从总体抽取样本容量为$$n$$的样本$$(X_1,X_2,...,X_n)$$,则子样中具有该特征的个体数目就是$$\sum_{i=1}^nX_i$$,样本均值$$\overline X = \frac{1}{n}\sum_{i=1}^nX_i$$ 称为样本比例。 该检验假设$$H_0: p=p_0,H_1: p \ne p_0$$; 当$$H_0$$为真时，$$\sum_{i=1}^nX_i$$服从二项分布$$B(np,np(1-p))$$, 根据中心极限定理，当样本容量$$n$$较大时，统计量$$U= \frac{\sum_{i=1}^nX_i-np_0}{\sqrt{np_0(1-p_0)}} = \frac{\overline X - p_0}{\sqrt{p_0(1-p_0)/n}}$$

近似服从正态分布$$（n>=30）$$，所以对总体比例的检验可选用正态总体已知方差的均值检验方法。

### **非参数的假设检验**

如果**已知总体分布的类型**，要推断的只是总体的某一个或某几个参数，称这类检验为参数检验，其他的检验称为非参数检验。

非参数检验中的假设可以是参数假设，也可以是非参数假设，常用的有符号检验法，秩和检验法，分布函数皮尔逊$$\chi^2$$检验法。

#### **皮尔逊**$$\chi^2$$**检验法**

**首先要假设总体服从某个给定的已知分布**：

若总体离散，则原假设$$H_0: X\sim P(X=x_i)=p_i$$. 若总体连续，则原假设$$H_0: X\sim f_0(x)$$

* 皮尔逊定理，设$$A_1,A_2,...,A_m$$是完备事件，$$P(A_i) = p_i$$,其中$$p_i$$已知，$$f_i$$是$$n$$次独立重复实验中$$A_i$$发生的次数，则当$$n \rightarrow \infty $$时 $$\sum_{i=1}^m(\frac{f_i}{n}-p_i)^2\frac{n}{p_i} = \sum_{i=1}^m\frac{(f_i-np_i)^2}{np_i} \sim \chi^2(m-1)$$

  该统计量含义如下：

关系如下：

| -- | $$A_1$$ | $$A_2$$ | ... | $$A_n$$ |
| :---: | :---: | :---: | :---: | :---: |
| 实际频数 | $$f_1$$ | $$f_2$$ | ... | $$f_n$$ |
| 理论频数 | $$np_1$$ | $$np_2$$ | ... | $$np_n$$ |
| 实际频率 | $$f_1/n$$ | $$f_2/n$$ | ... | $$f_n/n$$ |
| 概率 | $$p_1$$ | $$p_2$$ | ... | $$p_n$$ |

频率是以概率收敛于概率，如果$$H_0$$为真，当$$n$$越来越大时，频率与概率之间的差异将越来越小，因此，频率与概率之间的差异程度可以反映$$H_0$$是否成立。卡尔逊首先证明该统计量极限分布是自由度为$$n-1$$的卡方分布，因而在实用上，**当**$$n$$**很大** $$（n>=50）$$,可用卡方分布来确定拒绝域，即$$\sum_{i=1}^m\frac{(f_i-np_i)^2}{np_i} > \chi_{\alpha}^2(m-1)$$

* 皮尔逊定理的推广定理  

不论总体$$X$$服从什么分布，只要$$H_0: F(x)=F_0(x;\theta_1,\theta_2,...,\theta_r)$$为真，且未知参数是在$$H_0$$为真的情况下，用极大似然法得到的$$\hat \theta_i$$,则当$$n \rightarrow \infty $$时$$\sum_{i=1}^m\frac{(f_i-np_i)^2}{np_i} \sim \chi^2(m-r-1)$$

$$r$$是未知参数个数。

在使用该检验法时，需注意：1. 样本容量$$n$$要求很大，一般$$n \ge 50$$; 2. 划分小区间$$A_i$$，其个数$$m$$不宜太大，也不宜过小，实际应用中，一般取\[6,20\], 同时要求使每个小区间$$A_i$$上的$$np_i \ge 5$$最好大于$$10$$；

卡方检验的步骤:

1） 根据问题提出假设$$H_0$$  
2） 把实数轴划分为$$m$$个不相交的区间$$(a_i,a_{i+1}]$$,边界可以取无穷，视具体情况而定。  
3） 计算概率$$p_i = P(a_i<X<a_{i+1}) = F(a_{i+1})-F(a_i)$$,并计算出理论频数$$np_i$$  
4） 计算样本观测值$$x_1,x_2,...,x_n$$ 落在$$(a_i,a_{i+1}]$$的频数$$f_i$$ 

5） 计算统计量，给定显著水平，查临界值，进行判断

**皮尔逊**$$\chi^2$$**检验，列联表的独立性检验**

一般情况下，考察二元总体是否相关，$$（X，Y）$$

1.将$$X$$的指标取值范围分成$$m$$个互不相交的区间$$A_1,A_2,...,A_n$$;将$$Y$$的指标取值范围分成$$m$$个互不相交的区间$$B_1,B_2,...,B_n$$;记$$P\{X \in A_i,Y \in B_j\} = p_{ij}$$

2.$$X,Y$$是否相关，即$$H_0:$$$$X,Y$$是否相互独立

> 如果$$H_0$$ 为真，则应有 $$p_{ij} = p_{i.}*p_{.j}, i\in(1,2,...,m); j \in (1,2,...,k)$$，故原假设也可设为 $$H_0:p_{ij} = p_{i.}*p_{.j} $$ 此假设中有$$m+k$$个未知参数，但有两个约束条件，$$\sum p_{i.} = 1; \sum p_{.j} = 1$$;因而独立的未知参数有$$m+k-2$$个； 1. $$H_0$$为真时，用最大似然法估计未知参数，每个样本点落在$$(A_i,B_j)$$的概率是$$P\{X \in A_i,Y \in B_j\} = p_{ij}$$,落在$$(A_i,B_j)$$中的样本点个数为$$n_{ij}$$,他们是独立的，所以$$n_{ij}$$个样本点落在$$(A_i,B_j)$$的估计概率为$$p_{ij}^{n_{ij}}$$，因而似然函数$$L = \prod_{i=1}^m\prod_{j=1}^k p_{ij}^{n_{ij}} =^{H_0 is true} \prod_{i=1}^m\prod_{j=1}^k$$
>
> $$logL = {n_{m.}}log(1-\sum_{i=1}^{m-1}p_{i.})+{n_{.k}}log(1-\sum_{j=1}^{k-1}p_{.j})+\sum_{i=1}^{m-1}n_{i.}logp_{i.}+\sum_{j=1}^{k-1}n_{.j}logp_{.j}$$

> 得到似然方程,求偏导等于0 $$log_ax = \frac{1}{xlna}$$
>
> $$\frac{\partial logL}{\partial p_{i.}} = -\frac{n_{m.}}{1-\sum_{i=1}^{m-1}p_{i.}} + \frac{n_{i.}}{p_{i.}} = \frac{n_{i.}}{p_{i.}} - \frac{n_{m.}}{p_{m.}}  = 0$$
>
> 同理
>
> $$\frac{\partial logL}{\partial p_{.j}} = -\frac{n_{.k}}{1-\sum_{j=1}^{k-1}p_{.j}} + \frac{n_{.j}}{p_{.j}} = \frac{n_{.j}}{p_{.j}} - \frac{n_{.k}}{p_{.k}}  = 0$$
>
> 解得：
>
> $$\frac{n_{i.}}{p_{i.}} = \frac{n_{m.}}{p_{m.}} \Rightarrow p_{i.}(n_{1.}+n_{2.}+...+n_{m.}) = n_{i.}(p_{1.}+p_{2.}...p_{m.}) \Rightarrow \hat p_{i.} = n_{i.}/n$$
>
> 同理:
>
> $$\hat p_{.j} = n_{.j}/n$$

3. 构造检验统计量

$$\chi^2 = \sum_{j=1}^k\sum_{i=1}^m\frac{(n_{ij}-n\hat p_{ij})^2}{n\hat p_{ij}}$$

当$$H_0$$为真时：

$$\chi^2 = \sum_{j=1}^k\sum_{i=1}^m\frac{(n_{ij}-n\hat p_{i.}\hat p_{.j})^2}{n\hat p_{i.}\hat p_{.j}}$$

$$\chi^2 = \sum_{j=1}^k\sum_{i=1}^m\frac{(n_{ij}-n\frac{n_{i.}n_{.j}}{n^2})^2}{n\frac{n_{i.}n_{.j}}{n^2}} = n \sum_{j=1}^k\sum_{i=1}^m\frac{(n_{ij}-\frac{n_{i.}n_{.j}}{n})^2}{n_{i.}n_{.j}} \sim \chi^2(km-(m+k-2)-1)$$

给定显著水平$$\alpha$$;查的卡方分布表可得 $$\chi_{\alpha}^2((m-1)(k-1))$$ 使得

$$P\{\chi^2>\chi_{\alpha}^2((m-1)(k-1))\} \le \alpha$$

故得到$$H_0$$的拒绝域 $$\chi^2>\chi_{\alpha}^2((m-1)(k-1))$$

#### **符号检验法 （用于分位数检验）**

有些实际问题中对总体的分布不一定感兴趣，而只关心总体分布的分位数

$$eg$$ 关心99%的电视机的寿命是否不小于10000小时 ，即$$P(X\ge10000) = 0.99$$ ,把10000记为$$X_{0.99}$$称之为分位数。

* 单个总体的符号检验法

设总体$$X$$是连续型随机变量，密度函数为$$f(x)$$,$$x_p$$是其$$p$$分为点且$$f(x_p) > 0, P(X\ge x_p) = p$$,称$$x_p$$为$$X$$的$$p$$上侧分位数，$$P(X\le x_p) =\alpha$$为$$X$$的下侧分位数。

步骤：

> 1.提出检验问题 $$H_0: x_p \le x_0$$, 其中$$x_0$$为已知数，等价于概率问题$$P(X>x_0) \le p$$  
>
> 1\). 令$$\theta = P(X>x_0)$$, 当$$H_0$$为真时，有$$\theta=P(X_i>x_0) = P(X_i-x_0>0) \le P(X_i-x_p>0) = P(X>x_p) = p$$;  
>
> 2\). 由已知$$\theta \le p $$也能推出$$x_p \le x_0$$;证明如下：$$1-F(x_0) = P(X>x_0) = P(X_i>x_0) = P(X_i-x_0>0) \le P(X_i-x_p>0)=P(X_i>x_p)=1-F(x_p),F(x_0) \ge F(x_p)$$
>
> 2. 检验方法：因为每个样品只有两个可能，大于或小于$$x_0$$，所以该实验是伯努利概率，每个样本服从$$p=P(X\ge x_0)$$的\(0-1\)分布   
>
> 1\). 记$$n^+$$为样本中大于$$x_0$$的样品个数，则$$n^+ \sim B(N,p)$$; $$n^-$$是样本小于$$x_0$$的样品个数，亦服从二项分布，$$n^+ + n^- = N$$，$$n^0$$**是样本中等于**$$x_0$$**的个数，不研究**，所以$$N$$不一定是样本个数，不包括$$n_0$$的数目。  
>
> 2\). 选用$$n^+$$做检验统计量，显然$$n^+$$越小对$$H_0$$越有利，因为$$n^+$$越高，$$p$$分为数有可能越高，不利于$$H_0$$，因而$$n^+$$过高，则应该拒绝$$H_0$$，拒绝域的形式应取为$$W_0 = \{n^+ > C\}$$
>
> 3\). 对给定的显著水平$$\alpha$$,由二项分布定出$$C$$，易知$$C$$值是满足以下公式的最小整数$$P\{n^+ \ge C\} = \sum_{i=C}^N C_N^ip^i(1-p)^{N-i} \le \alpha$$

> 3. 故 $$H_0: x_p \le x_0$$,拒绝域 $$n^+ \ge C$$,C是满足$$\sum_{i=C}^N C_N^ip^i(1-p)^{N-i} \le \alpha$$ 最小整数；  
>
> $$H_0: x_p \ge x_0$$,拒绝域 $$n^+ \le C$$ $$C$$是满足$$\sum_{i=0}^C C_N^ip^i(1-p)^{N-i} \le \alpha$$最大整数;  
>
> $$H_0: x_p = x_0$$，拒绝域$$n^+ \ge C_2 or n^+ \le C_1$$，为双侧分位数判断，$$C_1$$是满足$$\sum_{i=0}^C C_N^ip^i(1-p)^{N-i} \le \alpha/2$$的最大整数；$$C_2$$是满足$$\sum_{i=C}^N C_N^ip^i(1-p)^{N-i} \le \alpha/2$$的最小整数。

* 两个总体的符号检验法

在做两个总体比较时，有时样本是成对出现的，此时该符号检验是以成对观察数据差的符号为基础，进行的检验，此时考虑是否有误差，即转换成两组是否同分布，从而$$\epsilon \le \phi $$ 与 $$\epsilon \ge \phi$$ 等可能，即差值小于$$0$$的概率为$$0.5$$，即差值的中位数是否为$$0$$，$$x_{0.5} = 0$$。故拒绝域为双侧分位数，$$n^+ \ge C_2 or n^+ \le C_1$$

**查表一般会给出**$$\sum_{i=0}^C C_N^i(1/2)^N \le \alpha/2$$**的最大C值，**$$n^+ + n^- = N$$**\(排除等于0的个体\)，那**$$C_2 = N - C_1$$**再判断是否位于拒绝域**

如果 $$N \gt 90$$ ,如何确定左侧分为数$$C_1$$;因为$$n^+ \sim B(N,p)$$,当$$N$$较大时，有中心极限定理，$$\frac{n^+-Np}{\sqrt{Np(1-p)}} \sim N(0,1)$$

给定$$\alpha$$后，查表得$$Z_{\frac{\alpha}{2}}$$,当位于拒绝域，则拒绝$$H_0$$,否则接受，即$$p=0.5$$

**符号检验是将非参数检验利用符号转化成参数性的检验问题，计算简单，不需要考虑总体分布，但因为遗弃了数据的真实值进行检验，精度较差\(**只考虑差数的正负号，而不考虑差数的绝对值差异，会导致部分试验信息损失，结果较为粗略\)

#### **秩和检验 （检验两个总体是否同分布）**

设$$x_1,x_2,...,x_n$$是两两互不相同的实数，若在$$x_1,x_2,...,x_n$$中恰有$$R_i$$个元素的值不超过$$x_i$$,则称$$x_i$$在$$x_1,x_2,...,x_n$$中的秩为$$R_i$$,若有几个实数相同，则用他们的平均值作为其秩。

> eg: 1.9,2.2,1.8,2.4,1.6,1.8; 其秩依次为4，5，\(2+\(2+1\)\)/2=2.5,6,1,2.5 累加为21 = 1+2+3+4+5+6

假定有两个相互独立的总体$$X$$和$$Y$$，他们分别由连续性分布函数$$F_X(x)$$和$$F_Y(y)$$均未知

> 1. $$H_0:F_X(x) = F_Y(y)$$,在两个总体中分别抽取$$m$$和$$n$$的独立样本，进行混合，其中$$Tx = \sum_{i=1}^mR_i$$ 为$$X$$子样在混合样本中的秩和；$$T_Y = \sum_{j=1}^nS_j$$ 为$$Y$$子样在混合样本中的秩和，$$T_x+T_y = 1+2+3+...+(m+n)$$
> 2. 当$$H_0$$成立时，根据对称性，$$P(X>Y) = P(Y>X) = 0.5$$;$$Tx$$\(样本数较小的那一方\)的取值不能太大也不能太小，**即太大和太小的那些值作为拒绝域**。可以查表得出秩和检验的结余表；
> 3. 当$$T_1<Tx<T_2$$则认为两个总体具有相同分布
> 4. 当$$m,n \le 10$$,通过查表得$$T1，T2$$的数值，当$$m,n \gt 10$$ 可利用$$T_x \sim N(\frac{m(m+n+1)}{2},\frac{mn(m+n+1)}{12})$$
> 5. 选取$$|(T_x-\mu)/\sigma | > Z_{\alpha/2}$$为拒绝域。
